# AgroSeva 🌿
### *Smarter Farming Starts Here*

AgroSeva is a state-of-the-art agricultural governance platform designed to bridge the gap between farmers and government support. By leveraging AI-driven verification and a transparent digital infrastructure, we empower the backbone of our economy with faster, more reliable access to essential resources.

---

## 🚀 The Vision
To revolutionize the agricultural landscape by digitizing service delivery, ensuring that every farmer—regardless of location—has equal and immediate access to government schemes, financial aid, and expert technical guidance.

## ⚠️ The Problem
Traditional agricultural support systems suffer from:
- **Opaque Processes**: Farmers often lack visibility into their application status.
- **Delayed Disbursements**: Verification can take months, missing critical farming seasons.
- **Disconnected Support**: A lack of centralized data leads to inefficient resource allocation.
- **Unresolved Grievances**: Complaints often get lost in bureaucratic layers.

## 💡 The AgroSeva Solution
AgroSeva addresses these challenges through a unified, high-tech ecosystem:

1.  **AI-Accelerated Verification**: Our AI service processes document checks and claim verifications within **48 hours**, replacing months of manual labor.
2.  **Unified Scheme Portal**: A one-stop-shop for 45+ active government schemes, featuring mobile-first document uploads for Aadhaar and Land Records.
3.  **Precision Agriculture**: Integration of satellite imagery and drone-based field analysis to provide data-driven insights to farmers.
4.  **SLA-Driven Grievance Engine**: A transparent tracking system that ensures issues are logged, assigned, and resolved with full accountability.

---

## ✨ Key Features

- **📊 Intelligent Dashboard**: Real-time tracking of applications and personalized scheme recommendations.
- **📱 Mobile-First Experience**: Designed for the field—snap and upload documents directly from a smartphone.
- **🤖 AI Document Processing**: Automated extraction and validation of land records and identification.
- **📡 Real-Time Monitoring**: Live updates on scheme disbursements and grievance resolution status.
- **🛡️ Secure & Private**: Bank-grade encryption protecting all personal and financial documents.

## 🛠️ Technology Stack

### **Frontend**
- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **State Management**: [TanStack Query](https://tanstack.com/query/latest)

### **Backend**
- **Runtime**: [Node.js](https://nodejs.org/) / [Express](https://expressjs.com/)
- **Database**: PostgreSQL (Prisma ORM)
- **Authentication**: JWT-based secure sessions

### **AI Service**
- **Language**: Python 3.10+
- **Framework**: Flask/FastAPI
- **Capabilities**: OCR (Optical Character Recognition), Automated Claim Verification, Image Analysis for Crop Monitoring.

## 👨‍💻 Getting Started

### Prerequisites
- Node.js (v18+)
- Python (v3.10+)
- PostgreSQL

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/kartik-suryawanshi/AgroSeva.git
   cd AgroSeva
   ```

2. **Setup Frontend**
   ```bash
   npm install
   npm run dev
   ```

3. **Setup Backend**
   ```bash
   cd backend
   npm install
   npm start
   ```

4. **Setup AI Service**
   ```bash
   cd ai-service
   pip install -r requirements.txt
   python app.py
   ```
